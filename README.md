## 🚀 Live Demo
[https://abumunaf-debug.github.io/ai-developer-projects/](https://abumunaf-debug.github.io/ai-developer-projects/)

## 📖 About
A fully functional task management application built as part of the **AI Developer Course (Octo Apps)**.

## ✨ Features

### Task Management
- ✅ Create, complete, and delete tasks
- 🏷️ Priority levels: Low, Medium, High
- 🔍 Filter tasks by status (All, Active, Completed)
- 📊 Real-time statistics
- 💾 Persistence with localStorage

### Book Search
- 📚 Search books via Open Library API
- ⏳ Loading states with spinner
- ❌ Error handling with retry button
- 📖 Import books as reading tasks

### Authentication & Routing
- 🔐 Login with student/instructor roles
- 🛡️ Protected routes with guards
- 🧭 Role-based navigation
- 👤 User profile view
- 🔑 Admin panel (instructor only)

## 🛠️ Tech Stack
- **JavaScript (ES Modules)**
- **Vite** - Build tool
- **GitHub Pages** - Hosting
- **GitHub Actions** - CI/CD
